很多人知道AI，但也只限于AI写个文章，做个图片或者做个视频。之所以局限，是因为**缺少系统的知识概念，无法区分产品定位和分类，甚至很多名词含义都不甚了解**。

### 为什么要掌握AI名词和解释？

首先你得知道它是什么，有什么含义，才能更好的评估和应用。

#### 方便沟通协作

- 避免因理解偏差导致错误和延误
- 准确的术语，有助于抹平技术鸿沟，让非技术人员也能理解

#### 提升学习效率

- 只有理解概念，才能深入学习更复杂的算法和架构
- 方便更快找到解决方案、评估工具，解决问题

### 为什么要理解和学习AI分类?

有助于理解和驾驭人工智能复杂领域的系统性框架。是理解AI能力边界、应用场景、发展阶段以及潜在影响的关键点。

**理解AI本质，能力边界**

- 区分弱AI，强AI，通用AI，理解当前绝大多数AI都属于弱AI，有助于避免对AI产生不切实际的幻想和担忧
- 区分计算智能、感知智能、认知智能等，帮助理解AI在不同层面的发展水平

**有助于AI技术选型与应用**

- 不同AI分类（如机器学习、深度学习、符号AI）适用于不同类型的问题（图像识别用卷积神经网络，自然语言理解用大型语言模型）
- 理解AI分类和应用，帮助发现未被发掘的AI应用场景，提供创业机会

**规划AI研发和学习路径**

- 有助于明确选择研究方向，并进行深入探索
- AI领域发展迅速，通过分类可以帮助系统的组织知识，了解分支发展、核心技术和应用前景等，进而构建自己的知识体系

**促进交流与理解**

- AI分类有一套通用的术语和概念，方便不同专业人士之间有效沟通
- 了解AI的分类和能力范围，可以更理性地看待AI发展情况

### 为什么要查看AI应用案例？

**识别真价值，找到AI真实的应用价值**

目前市场上充斥着大量AI宣传，自媒体渲染大肆渲染，以及企业为了AI而AI的产品。通过分析各行业的深度案例，可以更好的看清

**跨行业的技术启发和迁移**

创新往往发生在不同领域的交叉点。分析案例可以实现跨界思维复用
